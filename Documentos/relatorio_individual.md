# Tarefas Individuais

No âmbito do desenvolvimento do projeto NextBid, assumi a responsabilidade por componentes fundamentais da gestão estratégica e da fundamentação técnica do relatório. O meu contributo incidiu sobre a elaboração do Project Charter, a estruturação da WBS (Work Breakdown Structure) e o desenvolvimento das secções de Estatística e Programação Web.

No Project Charter, trabalhei na definição formal dos objetivos do projeto, identificação de stakeholders e alinhamento das expectativas gerais, garantindo que a visão do NextBid estivesse documentada de forma clara e rigorosa. Complementarmente, desenvolvi a WBS, onde decompus o projeto em pacotes de trabalho manejáveis. Esta estrutura foi essencial para organizar a hierarquia das tarefas e assegurar que todos os entregáveis, desde a base de dados até à interface do utilizador, estivessem devidamente mapeados.

Na vertente técnica, colaborei na secção de Programação Web, focando-me na arquitetura tecnológica da plataforma. Trabalhei na definição de como a interface deve interagir com o backend para proporcionar uma experiência fluida de licitação em tempo real. Além disso, na componente de Estatística, ajudei a estruturar a lógica de análise de dados que permitirá ao sistema gerar métricas relevantes, como a evolução de preços dos leilões e o comportamento dos utilizadores, transformando dados brutos em informação útil para a plataforma.

# Avaliação Crítica Pessoal

Globalmente, considero que os meus contributos foram determinantes para a organização e viabilidade técnica do NextBid. A criação da WBS, em particular, permitiu transformar uma ideia abstrata num plano de ação concreto, facilitando a distribuição de trabalho entre a equipa.

Uma das principais dificuldades encontradas foi a articulação entre a visão de alto nível do Project Charter e a especificidade técnica exigida na Programação Web. Foi necessário um esforço adicional para garantir que os requisitos técnicos definidos estivessem em total conformidade com os objetivos de negócio traçados inicialmente. Para superar este desafio, mantive um diálogo constante com os responsáveis pelas outras áreas técnicas para assegurar a coerência do sistema.
